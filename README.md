## Blockchain final project

主題： Gamefi  

核心 寵物養成遊戲  

主要相關合約會是： ERC721、ERC20

概念：  
1.最初會有一次的免費生成寵物   
2.在遊戲中提供一些任務：獲得代幣  
3.需使用代幣及道具nft來升級寵物：消耗代幣&道具nft
